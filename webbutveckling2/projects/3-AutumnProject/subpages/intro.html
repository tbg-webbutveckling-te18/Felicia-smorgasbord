If Else Statements

Now that you have linked your Js document with your HTML document we will go through the basics of if else statments.

For starters what is an if else statement? An if else statment is used to choose what will and what will not happen when given sertain information. With an if else statement you can check if the information you are given is true or false. It can example be if a specifik button is pressed something will happen, but if another button is pressed either something else or nothing will happen. It all depends on what you have writen in your code.

If Else Code Example

if (button == this){

document.writeln("do this");

}else {

document.writeln("do thit");

}

For-loops

A for-loop is a of an argument that is looped through (repeted) a number of times. For exampale you can loop through a list of ex. numbers 1-10. Maybe you want to print every number or print a number in a specific place in the list.

For-loops Code Example

for (var i = 0; i <= 10; i++) {

document.writeln(i);

}

Arrays

An Array is a kind of list that can contain different types of thing like variables and objects.

An Array has an index which means that the 1st object is on index 0 and the 10th object is on index 9.

Arrays Code Example

var kevinArr = ['Kevin', 24,'Seoul', 'Jacob'];

Objects

An object is a sort of variable that can contain all sorts of values. Unlike an array an object does not have an index.

Objects Code Example

var Kevin = {

name: 'Kevin',

age: 24,

city: Seoul,

friends: ['Jacob', 'Eric'],

isEvil: false,

}
